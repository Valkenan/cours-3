## Project Structure

Each lesson folder contains:
- `.cpp` file - Main C++ source code
- `.sln` file - Visual Studio solution file
- `.vcxproj` file - Visual Studio project file
- `.vcxproj.filters` file - Visual Studio project filters

## Building and Running

These projects are configured for Visual Studio. To run any lesson:

1. Navigate to the lesson folder
2. Open the `.sln` file in Visual Studio
3. Build and run the project (F5 or Ctrl+F5)

## Requirements

- Visual Studio (with C++ development tools)
